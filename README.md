# Smart File Organizer

A modern, intuitive desktop application built with Java and JavaFX that automatically organizes files based on predefined rules.


## ✨ Features

- **Intuitive UI:** A clean, modern user interface designed for ease of use.
- **In-Place Organization:** Creates organized subfolders (`PDFs`, `Images`, `Documents`, etc.) directly within the folder you select.
- **Real-Time Feedback:** A progress window appears during operation, letting you know the process is working.
- **Automatic Navigation:** Once complete, the application automatically opens the organized folder for you to view the results.
- **Recent History:** The app keeps a running list of the folders you've recently organized.

---

## 🛠️ Built With

- **Java & JavaFX:** Core application logic and user interface.
- **CSS:** Custom styling for a professional, modern look and feel.
- **jpackage:** To create a native Windows installer.

---

## 🚀 How to Install and Run

1.  Go to the [**Releases**](https://github.com/Bhudev-Berwal/SmartFileOrganizer/releases) section of this repository.
2.  Download the `SmartFileOrganizer-1.0.exe` installer from the latest release.
3.  Run the installer and follow the on-screen instructions. The application will be added to your Start Menu and Desktop.

That's it! No need to install Java or configure anything.
